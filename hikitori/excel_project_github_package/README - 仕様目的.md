---

## 主な機能

- Excelデータの一括入力・整形
- ファイル・フォルダの自動作成
- データ変換（Power Query）
- 未変換データの抽出・一覧化
- Pythonによる類似候補抽出（fuzzy matching）

---

## 処理フロー（概要）

1. Excelシートに入力
2. VBAで前処理・ファイル操作
3. Power Queryでデータ変換
4. 未変換データを抽出
5. Python（PY関数）で類似候補を生成
6. 結果をExcelに出力

詳細は `/docs/workflow.md` を参照

---

## 各要素の役割

### VBA
- 入力処理
- ファイル作成
- フロー制御
- イベント処理（ThisWorkbook）

### Power Query
- データ結合・変換
- 未変換データの抽出

### Python（Excel内）
- 類似検索
- 候補リスト生成

---

## 注意事項

- VBAは `.bas / .cls / .frm` として管理
- フォームは `.frm` と `.frx` をセットで管理
- Excel本体（.xlsm）は参考用であり、コードは本リポジトリ内で管理する
- シート構成は `/workbooks_summary/` 配下の `_sheets.md` を参照
